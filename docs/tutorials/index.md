# SreeBase Tutorial Roadmap

This tutorial is written for new learners. You do not need to know database internals before starting.

## Learning Path

| Step | Lesson | Goal |
| --- | --- | --- |
| 1 | [Installation](installation.md) | Run the server and CLI |
| 2 | [First Database](first-database.md) | Insert and read your first documents |
| 3 | [Core Concepts](concepts.md) | Understand collections, documents, indexes, and roles |
| 4 | [Insert Data](insert.md) | Add records safely |
| 5 | [Read Data](read.md) | Filter, sort, and limit results |
| 6 | [Update Data](update.md) | Change matching documents |
| 7 | [Delete Data](delete.md) | Remove documents carefully |
| 8 | [Aggregations](aggregations.md) | Group and calculate summaries |
| 9 | [Indexes](indexes.md) | Speed up equality lookups |
| 10 | [Python SDK](../api.md) | Build an app with `reddybase` |

## What SreeBase Teaches

- How a database stores records in files.
- How a lexer and parser turn text into executable commands.
- How indexes trade memory for faster reads.
- How a TCP server can expose a database over the network.
- Why security and validation matter around query languages.

## Before You Start

You should have:

- Python 3.11 or newer
- A terminal
- Basic comfort with Python dictionaries
